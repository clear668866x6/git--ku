=== 测试用例9：break 和 continue ===
int main(void) {
    int i = 0;
    int sum = 0;
    for (i = 0; i < 10; i = i + 1) {
        if (i == 3)
            continue;
        if (i == 8)
            break;
        sum = sum + i;
    }
    write(sum);
    return sum;
}
=== 输入: (无) ===
=== 预期输出: 25 === (0+1+2+4+5+6+7=25, skip 3, break at 8)
=== 预期返回值: 25 ===
